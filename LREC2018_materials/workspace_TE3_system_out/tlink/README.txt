This folder contains the stats for TE3 systems on temporal relation detection and classification
There 2 types of folder:

- systemName1-systemName2: compare 2 systems output for each of the 3 types of TLINK taken into account: event-event; event-timex and event-dct.

files which contain “detc” in their name focuses the data on the existence or not of TLINK
files which contain  “class” in their name focuses the data on the classification of a TLINK

- tlink_full: for each system (6 systems overall) it compares the GOLD tlink in terms of detection and classification
